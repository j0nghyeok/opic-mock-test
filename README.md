# OPIc 모의고사 (OPIc Mock Test)

브라우저에서 바로 돌아가는 **OPIc 스피킹 모의고사** 앱입니다. 실제 OPIc 구조(자기소개 → 서베이/돌발 콤보 → 롤플레이 → 어드밴스, 총 15문항)에 맞춰 문제를 읽어주고(TTS), 답변을 녹음하고, 녹음을 텍스트로 전사해 복습할 수 있습니다.

## 바로 사용하기

👉 **https://j0nghyeok.github.io/opic-mock-test/**

설치 없이 위 링크로 접속하면 됩니다. 마이크 권한을 허용하세요.

## 기능

- **실제 OPIc 구조 15문항** — 배경 서베이 선택(거주/여가/취미/운동/여행)에 맞춘 맞춤 출제
- **문제 읽어주기(TTS)** — 브라우저 음성, OpenAI 키 입력 시 자연스러운 AI 음성(nova)
- **다시듣기** — 문항당 1회, 리딩 종료 5초 후 비활성
- **자동 녹음** — 문제 리딩이 끝나면 자동으로 녹음 시작
- **결과 화면** — 문항별 녹음 재생/다운로드, 자가 채점 메모
- **🎤 전사본 다운로드** — 녹음을 텍스트로 변환해 문제와 함께 저장
  - OpenAI 키가 있으면 Whisper로 자동 전사 → `.txt`
  - 키가 없으면 오디오+문제 묶음 `.json` 저장 → 로컬 Whisper 등으로 전사
- **🤖 Claude 동적 문제생성(옵션)** — Anthropic API 키 입력 시 매 시험마다 Claude가 새 문제 생성 (미입력 시 내장 문제은행 사용)

## API 키에 대하여

OpenAI / Anthropic API 키는 **브라우저 localStorage에만 저장**되며, 이 저장소 코드에는 어떤 키도 포함되어 있지 않습니다. 키는 각자 발급해서 입력합니다(선택). 키 없이도 내장 문제 + 브라우저 음성으로 모든 핵심 기능이 동작합니다.

## 파일 구조

- `index.html` — 실행 파일 (GitHub Pages가 서빙)
- `shell.html` — 편집 원본 (`index.html`은 이 파일의 복사본)
- `questions.js` — 내장 문제 은행
- `PROGRESS.md` — 개발 진행/설계 기록

## 로컬 실행

```bash
python3 -m http.server 8000
# http://localhost:8000/ 접속
```

`file://`로 직접 열어도 대부분 동작하지만, Claude 문제생성 등 외부 API 호출은 CORS로 막힐 수 있어 로컬 서버 또는 위 배포 URL 사용을 권장합니다.
